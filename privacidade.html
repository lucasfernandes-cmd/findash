Política de Privacidade

Última atualização: 13 de março de 2026

O FinDash ("nós", "nosso" ou "aplicativo") é um dashboard de organização financeira pessoal e empresarial. Esta política descreve como coletamos, usamos e protegemos seus dados.

1. Dados coletados

Coletamos os seguintes dados quando você utiliza o FinDash:

  • Dados de cadastro: nome, e-mail, telefone e nome da empresa (opcional).
  • Dados financeiros: contas bancárias, cartões de crédito, transações, contas a pagar, dívidas e valores a receber que você cadastra voluntariamente.
  • Imagens enviadas para OCR: fotos de notas fiscais, extratos e comprovantes enviados para reconhecimento automático de texto.
  • Preferências do app: tema visual (claro/escuro) e modo ativo (empresa/pessoal).

2. Como usamos seus dados

  • Autenticação: e-mail e senha são usados para criar e proteger sua conta via Supabase Auth.
  • Funcionalidade do app: seus dados financeiros são armazenados para exibir dashboards, relatórios e categorias.
  • OCR com IA: imagens enviadas são processadas pela API Google Gemini para extração de texto. As imagens são transmitidas temporariamente e não são armazenadas pelo Google Gemini após o processamento.
  • Sincronização: seus dados são sincronizados entre dispositivos via Supabase Cloud.

3. Armazenamento e segurança

  • Dados são armazenados no Supabase (infraestrutura AWS) com criptografia em trânsito (TLS/HTTPS) e em repouso.
  • Implementamos Row Level Security (RLS) para garantir que cada usuário só acesse seus próprios dados.
  • Senhas são gerenciadas pelo Supabase Auth e nunca armazenadas em texto claro.
  • O app utiliza Content Security Policy (CSP) e outros headers de segurança para proteção contra ataques.

4. Compartilhamento de dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros para fins de marketing ou publicidade.

Seus dados são compartilhados apenas com:

  • Supabase: para autenticação e armazenamento de dados (como processador de dados).
  • Google Gemini API: imagens são enviadas temporariamente para processamento de OCR. Nenhum dado é retido após o processamento.

5. Seus direitos

Você tem direito a:

  • Acessar todos os seus dados diretamente no app.
  • Corrigir seus dados pessoais a qualquer momento via tela de perfil.
  • Excluir quaisquer registros financeiros ou sua conta completa.
  • Exportar seus dados em formato legível.

6. Cookies e armazenamento local

O FinDash utiliza localStorage do navegador para armazenar dados localmente como cache, melhorando a velocidade de carregamento. Não utilizamos cookies de rastreamento ou publicidade.

7. Menores de idade

O FinDash não é destinado a menores de 18 anos. Não coletamos intencionalmente dados de menores.

8. Alterações nesta política

Podemos atualizar esta política periodicamente. Alterações significativas serão comunicadas pelo app.
